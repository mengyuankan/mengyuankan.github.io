Set up Your Blog

Summary

Set up a relatively independent blog using the combination of GitHub and Hexo:

  • Build GitHub repository
  • Install and set up Hexo
  • Create and update posts

Build GitHub repository

  • Create GitHub account
  • Create GitHub repository by the name “yourname.github.io”
  • Download and install GitHub for windows
  • Use the repository in GitHub Desktop by clicking the icon

Install and set up Hexo

  • Download and install Node.js

  • Install Hexo in Git shell

  • Configurate Hexo

    • Find and open _config.yml in Hexo directory

    • Edit contents under the tag “Deployment”

    • Add updated file in GitHub repository

    • Choose and use theme “very-simple”

  • Configurate theme

    • open _config.yml under Hexo directory and edit

    • Open _config.yml under Hexo\themes\very-simple and edit

  • Update changes

Create and update posts

  • Create new post

  • Create customized page

  • Edit post using markdown

  • Update posts
