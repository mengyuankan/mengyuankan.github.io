Make Slides Using LaTex

2016-02-10
Tech

Make slides using LaTex document class Beamer:

  • Install LaTex
  • Install Texmaker
  • Create slides using Beamer

Install LaTex

  • Download MiKTeX, the implementation of LaTex
    • Choose “Other Downloads: MiKTeX Net Installer”
  • Download complete MiKTex
    • Click MikTex -> Choose “Download MikTex” -> Choose “Complete MikTex” -> Save in a specific folder (This process will take a while and more than 1 Gb space)
  • Install MikTex
    • Click *.exe in the download folder and install by default
  • Check LaTex in windows command prompt

Install Texmaker

  • Download and install Texmaker (executable version)
  • Configure Texmaker
    • Open Texmake -> Click “Configure Texmaker” in the menu “Options” -> In TEX Command, choose “Embed” in “Pdf Viewer” -> In TEX Editor, choose “ISO-8859-1” in “Editor Font Encoding”, choose “back up documents in every 10 min”, choose “en_US.dic” in “Spelling dictionary” and leave other things by default

Create slides using Beamer

  • Clink “Wizard” in the menue -> Choose “Quick Beamer Presentation” -> Choose a templete and edit -> Click “Quick Build” arrow

Tips

  • Only .tex file is needed in all the files generated
